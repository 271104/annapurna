## 🚀 Quick Start Deployment:

### Option 1: Fastest (Using Vercel CLI)
```bash
# Install Vercel CLI
npm install -g vercel

# Deploy frontend
cd frontend
vercel login
vercel --prod
```

### Option 2: Recommended (Using GitHub)
1. Push code to GitHub
2. Deploy backend on Render.com
3. Deploy frontend on Vercel.com
4. Update environment variables
5. Create admin account

**Detailed instructions in**: `DEPLOYMENT_GUIDE.md`
