# 📸 Images Folder

## Place Your Images Here

### Required Images:

1. **logo.jpg** (or logo.png)
   - Your business logo
   - Size: 200x200px recommended
   - Used in: Navbar

2. **shop-front.jpg**
   - Your shop/store photo
   - Size: 1920x1080px recommended
   - Used in: Hero section, About section

### Optional Images:

3. **gallery-1.jpg to gallery-6.jpg**
   - Additional photos
   - Size: 800x800px recommended
   - Used in: Gallery section

---

## Quick Start

1. Copy your images to this folder
2. Name them as shown above
3. Refresh browser
4. Done! ✅

---

## Supported Formats

- ✅ .jpg / .jpeg
- ✅ .png
- ✅ .webp
- ✅ .gif
